# 學生申訴流程法規查核輔助系統

版本：v0.5.2-hotfix

## 使用狀態

本工具目前為人工試用候選版，尚未作為正式行政依據。  
上線前與正式使用前，仍應由承辦單位依最新法規、主管機關函釋、地方規範與校內學生申訴評議規定進行人工複核。

## 定位

本系統為靜態程序查核輔助工具，協助承辦人檢查學生申訴流程、期限、申評會組成、表件提醒與程序風險。

## 使用邊界

- 不儲存個案資料。
- 不登入。
- 不上傳。
- 不使用 AI 問答。
- 不取代申評會判斷。

## 主要頁籤

- 總覽
- 一般申訴
- 再申訴
- 專法分流
- 申評會組成
- 表件與風險
- 上線前驗收

## 注意

上線前仍應由承辦單位依最新法規、主管機關函釋、地方規範與校內規定人工複核。
